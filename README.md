# FFT
Freaky File Transfer, an UDP file transfer written in python

## Version
The software is written and tested in python 3.10.5

## User guide
### Run a server
```
python server.py
```
or if you still have python 2.x versions
```
python3 server.py
```

### Run a client
```
python client.py
```
or if you still have python 2.x versions
```
python3 client.py
```
